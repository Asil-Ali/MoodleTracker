## ⚙️ الإعداد — خطوة بخطوة

### الخطوة 1: تثبيت Python
تأكدي أن Python مثبت على جهازك:
```
python --version
```
لو ما موجود: حمّليه من https://python.org

### الخطوة 2: تثبيت المكتبات
```bash
pip install requests beautifulsoup4 lxml
```

### الخطوة 3: إنشاء Telegram Bot

1. افتحي Telegram وابحثي عن **@BotFather**
2. أرسلي: `/newbot`
3. اختاري اسماً (مثلاً: Aseel Moodle Bot)
4. اختاري username (مثلاً: aseel_moodle_bot)
5. ستحصلي على **Token** — انسخيه

**للحصول على Chat ID:**
1. افتحي محادثة مع بوتك الجديد وأرسلي أي رسالة
2. افتحي هذا الرابط في المتصفح (غيّري YOUR_TOKEN):
   ```
   https://api.telegram.org/botYOUR_TOKEN/getUpdates
   ```
3. ابحثي عن `"chat": {"id": XXXXXXX}` — الرقم هو Chat ID

### الخطوة 4: تعديل config.py

```python
STUDENT_ID = "2022XXXXXX"        # رقمك الجامعي
PASSWORD   = "your_password"     # كلمة المرور (نفسها في Moodle)

TELEGRAM_TOKEN   = "1234567:ABCdef..."
TELEGRAM_CHAT_ID = "987654321"
```

### الخطوة 5: اختبار

```bash
python main.py --test
```
لو وصلتك رسالة على Telegram ✅ كل شيء تمام

### الخطوة 6: تشغيل

```bash
python main.py
```
